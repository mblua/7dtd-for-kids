# 7 Days to Die — For Kids

A quick and basic mod that attempts to make 7 Days to Die less terrifying for children. This is not a polished mod — an adult should test it first before letting kids play.

## What this mod does

- **Zombies are now colored cylinders.** Each zombie type has a unique color (blue, red, green, etc.) so you can tell them apart. The original zombie models are hidden behind the cylinders.
- **Hitboxes remain unchanged.** Damage is still registered on the original (invisible) zombie hitbox, not on the visible cylinder. This means you'll need to learn where to aim for each colored cylinder — it takes some getting used to.
- **Loading screens are blacked out.** The original loading screen images were too scary, so they've been replaced with a plain black screen.
- **Gore blocks replaced with rubble.** Gore piles, hanging corpses, and similar decorations are replaced with debris/rubble models.
- **Blood-on-hit replaced with sparks.** Hitting zombies produces stone sparks instead of blood particles. Dismemberment is disabled.
- **Zombie names are kid-friendly.** Names like "Arlene the Blue", "Silver", etc. replace the original names.
- **Sounds are unchanged.** The original game sounds remain as-is.

## Installation

1. Copy the `Mods/7DaysForKids/` folder into your game's `Mods/` directory
2. Launch the game **without EAC** (Easy Anti-Cheat)
3. The mod loads automatically

## Known limitations

- The colored cylinders don't perfectly match the invisible hitbox — aiming takes practice
- Zombie animals (zombie bear, zombie dog, zombie vulture) are also cylinders but sized differently
- Sounds are still the original ones (screams, growls, etc.)
- Some zombie body parts may poke through the cylinder in debug mode
- Loot drops as a rubble block on the ground where the zombie died, not from the cylinder itself

# 7 Days to Die — Para Chicos

Un mod rapido y basico que intenta hacer 7 Days to Die menos terrorifico para los chicos. No es un mod muy pulido — lo deberia probar un adulto primero antes de dejar que jueguen los chicos.

## Que hace este mod

- **Los zombies ahora son cilindros de colores.** Cada tipo de zombie tiene un color unico (azul, rojo, verde, etc.) para poder distinguirlos. Los modelos originales de los zombies estan ocultos detras de los cilindros.
- **Los hitboxes no cambiaron.** El dano se registra en el hitbox original (invisible) del zombie, no en el cilindro visible. Esto significa que hay que aprender a donde pegarle a cada cilindro de color — lleva un poco de practica.
- **Las pantallas de carga estan en negro.** Las imagenes originales de las pantallas de carga daban miedo, asi que se reemplazaron con una pantalla negra.
- **Los bloques de gore se reemplazaron con escombros.** Pilas de gore, cadaveres colgantes y decoraciones similares se reemplazaron con modelos de escombros.
- **La sangre al golpear se reemplazo con chispas.** Golpear zombies produce chispas de piedra en vez de particulas de sangre. El desmembramiento esta desactivado.
- **Los nombres de los zombies son aptos para chicos.** Nombres como "Arlene the Blue", "Silver", etc. reemplazan los nombres originales.
- **Los sonidos no se cambiaron.** Los sonidos originales del juego se mantienen como estan.

## Instalacion

1. Copiar la carpeta `Mods/7DaysForKids/` en el directorio `Mods/` del juego
2. Iniciar el juego **sin EAC** (Easy Anti-Cheat)
3. El mod se carga automaticamente

## Limitaciones conocidas

- Los cilindros de colores no coinciden perfectamente con el hitbox invisible — apuntar requiere practica
- Los animales zombie (zombie bear, zombie dog, zombie vulture) tambien son cilindros pero de diferente tamano
- Los sonidos siguen siendo los originales (gritos, gruñidos, etc.)
- Algunas partes del cuerpo del zombie pueden asomarse por el cilindro en modo debug
- El loot aparece como un bloque de escombros en el suelo donde murio el zombie, no sale del cilindro
